# Genesis Ember
[Play Genesis: Ember](https://sarahemmelin.github.io/GenesisGame/)

Genesis Ember is an interactive browser based simulation where players observe and interact with genetic inheritance. The player manages a population of glowing organisms called Embers. By selectively breeding these Embers, the player can observe concepts like genetic drift, population bottlenecks, epistasis, and the founder effect.

## Teacher Aid
[View the Teacher's Aid](https://sarahemmelin.github.io/GenesisGame/teachers-aid.html)

This repository also contains an educational guide designed to help teachers use this game in the classroom. The guide includes detailed explanations of the genetic mechanics, walkthroughs of the specific directed learning modes, and suggested questions to facilitate class discussion.
